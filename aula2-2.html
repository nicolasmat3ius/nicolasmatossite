Entradas (LDR)

Este projeto demonstra o uso de uma entrada analógica (LDR) para controlar uma saída digital (LED) em um Arduino. O LDR (Light Dependent Resistor) é um sensor que muda sua resistência de acordo com a quantidade de luz incidente. Neste exemplo, o LED acenderá quando a luz ambiente estiver abaixo de um determinado nível.

   const int LED = 13;
   int LDR = A0;
   void setup() {
    pinMode (LED,OUTPUT);
    pinMode (LDR,INPUT);
    Serial.begin(9600);
   }
   void loop () {
  int luz = analogRead (LDR);
  if (luz>500){
    digitalWrite(LED,HIGH);
  }
  else {
    digitalWrite(LED,LOW);
  }
   }

    Explicação do código

    1. O LED é definido como uma saída digital no pino 13 do Arduino.
    2. O LDR é definido como uma entrada analógica no pino A0 do Arduino.
    3. No setup(), os pinos do LED e do LDR são configurados como saída e entrada, respectivamente. A comunicação serial é iniciada com uma taxa de 9600 bits por segundo.
    4. No loop(), o valor analógico do LDR é lido e armazenado na variável luz.
    5. Se o valor de luz for maior que 500 (indicando baixa luminosidade), o LED é ligado (HIGH).
    6. Caso contrário, o LED é desligado (LOW).
