        // Código em C#
        void setup() {
            pinMode(2, INPUT);
            pinMode(13, OUTPUT);
        }
        
        void loop() {
            if (digitalRead(2) == HIGH) {
                digitalWrite(13, HIGH);
            } else {
                digitalWrite(13, LOW);
            }
        }

    Explicação

    Neste circuito, conectamos um LED a um Arduino. Utilizamos um botão como entrada para o Arduino. Quando o botão é pressionado, o Arduino lê um sinal de entrada alto e acende o LED.
